# WalletAPI 

## Configuration (Required)
To run the API, inside the folder database/variables.js password and username must be created.

### Example database/variables.js file

    const  userName  =  <username>;
	const  password  =  <password>;

	  
	module.exports  = {
		userName,
		password
	};

## Installation
    npm install

## Run
    node app.js
